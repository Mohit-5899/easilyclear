## Bugs found

### 🔴 Bug #1 — Brand leak in canvas root label (FIXED in-flight)

**Severity:** High (visible brand leak, contradicts spec).
**Where:** `/library/rajasthan_geography` radial canvas, central node label.
**What:** Root node text rendered "Springboard Academy — Raja…" rotated vertically.
**Repro:** Open `/library/rajasthan_geography` → look at the central node of the radial canvas → the label text shows publisher name.
**Root cause:** `database/skills/rajasthan_geography/SKILL.md` `name:` field still held `Springboard Academy — Rajasthan Geography` from the v2 → v3 migration. Migration script preserved the original name verbatim instead of substituting the subject-canonical name.
**Fix applied during sweep:** Changed `name:` → `Rajasthan Geography` in `database/skills/rajasthan_geography/SKILL.md`.
**Evidence:**
- Before: `docs/qa/screenshots/07-library-subject.png`
- After: `docs/qa/screenshots/07b-library-after-fix.png`
**Follow-up:** the migration script (`scripts/migrate_to_subject_canonical.py`) should override the root `name` to a brand-free string going forward. Open as a small follow-up PR.

### 🟡 Bug #2 — Topic-picker breadcrumb duplicates subject name

**Severity:** Low (cosmetic, no data wrongness).
**Where:** `/tests` → "+ New test" modal → each leaf row's secondary line.
**What:** Breadcrumb reads `Rajasthan Geography · Rajasthan Geography › Physical Geography › Aravalli` — the subject name appears twice (once as `book_name` from the manifest, once as the first segment of the chapter `path`).
**Repro:** `/tests` → click `+ New test` → look at any leaf row's secondary line.
**Root cause:** `frontend/src/app/api/library/leaves/route.ts` `collectLeaves` walks from the subject root, so `path` already starts with the subject. The frontend renders `${book_name} · ${path}`, doubling it.
**Fix:** either drop `book_name` from the rendered line (it's redundant when there's only one subject) OR slice the first segment off `path` before rendering. Both ~3 lines of TypeScript.
**Evidence:** `docs/qa/screenshots/14-tests-modal.png`
