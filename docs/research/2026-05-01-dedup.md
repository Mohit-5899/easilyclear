# Cross-Book Deduplication — Research & Decision

**Date:** 2026-05-01
**Scale:** ~30-50 leaves/book × 3-5 books → ~3k-12k cross-book pairs.

## 1. Options

| Approach | Quality | Speed | Cost |
|---|---|---|---|
| Embed cosine `all-MiniLM-L6-v2` local | Medium (FPs on "related") | <1s | $0 |
| Embed cosine `bge-small-en-v1.5` local | High on textbook prose | ~5s | $0 |
| LLM judge every pair (Gemma 4 26B) | Highest, 3-class | 1-7h | $3-12 |
| **Hybrid: embed prefilter → LLM confirm** | **Highest, calibrated** | **~5min** | **~$0.10** |

Empirical thresholds (SemHash default 0.90; NeMo SemDedup ~0.92; arxiv:2410.01141 found 0.85-0.92 sweet spot for paper-title dedup):
- `< 0.80` different → skip
- `0.80-0.92` candidate → LLM-judge
- `≥ 0.92` auto-duplicate

## 2. DECISION

**Algorithm:** Hybrid. Encode all leaves with `bge-small-en-v1.5` (local, 33M params, 384-dim — beats MiniLM on MTEB and handles textbook prose well). Compute all cross-book pair cosines. Auto-mark duplicates at `≥ 0.92`; send the `0.80-0.92` grey zone to Gemma 4 26B as a 3-class judge ("duplicate" / "related" / "different").

**Winner-selection** (deterministic per cluster):
1. **Source authority**: NCERT (3) > RBSE Class 11/12 (2) > Springboard coaching (1).
2. Tie-break: longer body.
3. Final tie-break: smaller `node_id` (stable).

Losers get `superseded_by: <winner_node_id>` in frontmatter; retrieval filters them by default but keeps them indexed for citation.

## 3. Pseudo-code

```python
leaves = load_all_leaves()
embs = bge_small.encode([l.body for l in leaves], normalize=True)
uf = UnionFind(len(leaves))
for i, j in cross_book_pairs(leaves):
    cos = embs[i] @ embs[j]
    if cos < 0.80: continue
    if cos >= 0.92 or llm_judge(leaves[i].body, leaves[j].body) == "duplicate":
        uf.union(i, j)
for cluster in uf.groups():
    winner = max(cluster, key=lambda x: (AUTHORITY[x.source], len(x.body), -ord_id(x)))
    for loser in cluster - {winner}:
        set_frontmatter(loser, superseded_by=winner.node_id)
```

## 4. Cost

- Embeddings: local CPU, ~150 leaves → ~5s, **$0**.
- All-pairs cosine: 150² / 2 ≈ 11k dot products, **<100ms**.
- LLM judge: ~50-150 grey-zone pairs × ~500 tokens via OpenRouter Gemma → **$0.05-0.20**, ~5min wall.
- **Total: <$0.25, <10min.** Idempotent — cache embeddings by content hash.
